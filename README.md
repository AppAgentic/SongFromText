# SongFromText

Turn pasted messages into AI-generated songs using their exact words as lyrics.

> Paste what they said. Hear it as a real song.

## Status
Pre-scaffold. See [`CLAUDE.md`](CLAUDE.md) for agent guidance and [`docs/PRD.md`](docs/PRD.md) for the full product spec.
